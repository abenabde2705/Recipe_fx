Projet de Traitement de Recettes
Ce projet a été développé individuellement par Ben Abdessalem Amrou dans le cadre d'un projet personnel de traitement de recettes utilisant JavaFX.

Configuration
Le projet a été configuré et développé en utilisant les outils suivants :

Java Development Kit (JDK) version [version]
JavaFX [version]
IntelliJ IDEA [version] (IDE)
Maven [version] (gestionnaire de dépendances)
Toutes les dépendances nécessaires ont été correctement configurées dans le fichier pom.xml.

Problèmes Rencontrés
Redimensionnement des Images de Fond
Un des problèmes rencontrés lors du développement concerne le redimensionnement des images de fond dans l'interface utilisateur JavaFX. Malgré plusieurs tentatives et recherches, la solution optimale n'a pas encore été trouvée. Les images peuvent apparaître déformées ou mal redimensionnées sur certaines tailles d'écran.

Problème avec l'Exécutable
Un autre problème rencontré concerne la création de l'exécutable JAR de l'application. Bien que le projet se compile correctement et fonctionne sans problème dans l'IDE, l'exécution de l'exécutable JAR peut entraîner des erreurs liées à la configuration de JavaFX ou à d'autres problèmes d'environnement.

Remarque Importante
Malgré ces problèmes, le projet est fonctionnel dans l'environnement de développement. Il est recommandé d'exécuter le projet directement depuis l'IDE pour éviter les problèmes potentiels liés à la configuration de l'environnement d'exécution.
